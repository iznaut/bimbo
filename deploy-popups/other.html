one-click deployment via SFTP

most web hosting providers support SFTP (Secure File Transfer Protocol) as an option for uploading files. if your provider supports SFTP, you can configure deployment here.
if you're using a provider that supports file uploads via an API (Application Programming Interface), please email me at bimbo@iznaut.com and i'll look into adding Official Support for it in the future!

provider name can be anything - just determines how the menu item appears (e.g. "deploy to My Cool Webhost")

an URL-like address to your provider (e.g. sftp.mycoolwebhost.com)

port 22 is default for SFTP. don't change this unless your provider uses a different number

path to your site's root directory on the hosting server (e.g. /home/public)

your SFTP-enabled username (may or may not be different than what you use to login your host's web interface)

if disabled, you will be prompted for your SFTP password before each deployment
